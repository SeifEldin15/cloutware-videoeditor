Available Animation Styles

1. GIRLBOSS ANIMATION

Perfect for: Beauty, lifestyle, empowerment content

Description

Feminine, empowering style with customizable colors and smooth word-by-word reveals with glow effects.

Font

Luckiest Guy (Bold, Impact style)

Parameters

ParameterTypeDefaultDescription
subtitleStylestring-"girlboss"
fontFamilystring-"Luckiest Guy"
fontSizenumber48Font size (10-72)
girlbossColorstring"#FF1493"Text color (HEX)
girlbossShadowStrengthnumber1.0Shadow intensity (0.5-3.0)
girlbossAnimationstring"none""none" or "shake"
verticalPositionnumber18Position from bottom % (0-100) - UNIVERSAL
outlineWidthnumber2Text outline width (0-8)
outlineColorstring"#000000"Outline color (HEX)
outlineBlurnumber0Outline blur effect (0-10)

Example Usage

{
  "subtitleStyle": "girlboss",
  "fontFamily": "Luckiest Guy",
  "fontSize": 48,
  "girlbossColor": "#FF1493",
  "girlbossShadowStrength": 2.0,
  "girlbossAnimation": "shake",
  "verticalPosition": 18
}

Best Practices

Content Matching

  • Business/Sales → Hormozi style
  • Lifestyle/Beauty → Girlboss style
  • Professional → ThinToBold or Basic
  • Entertainment → WavyColors style

Font Selection Guidelines

  • Impact content → Luckiest Guy
  • Professional content → Montserrat Thin
  • Standard content → Arial

Available Fonts

Arial, Arial Black, Montserrat (Thin, Regular, ExtraBold, Black), Luckiest Guy, Impact, Helvetica, Georgia, Times New Roman, Verdana, Trebuchet, Comic Sans MS, Courier New, Garamond, Palatino Linotype, Bookman Old Style, Erica One, Bungee, Sigmar, Sora, Tahoma, Gotham Ultra, Bodoni Moda

Troubleshooting

Common Issues

  • Font Not Loading: Verify spelling, fallback fonts used automatically
  • Colors Not Displaying: Use HEX format (#RRGGBB), verify valid codes
  • Performance Issues: Reduce shadow strength, use simpler animations for long content
